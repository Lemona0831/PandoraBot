## 1.0 전투 방향

PANDORA 1.0은 전투를 **세션 단위로 관리**한다.

다만 1.0은 턴제 자동 전투 시스템이 아니며, 진행자의 운영 판단을 보조하는 전투 운영판을 목표로 한다.

## 1.0 포함 명령

- `/전투시작`
- `/전투종료`
- `/전투참가`
- `/에너미소환`
- `/전투상태`
- `/전투피해`
- `/전투회복`
- `/전투퇴장`
- `/전투정리`
- `/전투보조`

## 1.0에서 제외되는 요소

- 턴 관리
- 라운드 관리
- 행동완료 체크
- 이니셔티브
- 상태이상 자동 처리
- 성장 자동화
- 웹-봇 완전 연동

즉, 1.0은 세션을 열고 닫고, 참가자와 HP와 메모와 로그를 관리하는 범위까지를 다루고,
